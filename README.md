## Cloud Native Remote Sensing with Python

###### Course Materials found here: https://courses.spatialthoughts.com/python-remote-sensing.html

> A structured introduction to XArray, DuckDB, STAC, and Dask for cloud-based remote sensing applications.

This is an intermediate-level course that covers tools and techniques for working with climate and earth observation datasets using a modern cloud-native approach.

With the growing ecosystem of cloud-native data formats, open data catalogs, and powerful open-source packages, remote sensing practitioners are now able to adopt open and vendor agnostic cloud-based processing workflows. 

This class will cover how to implement these workflows using Python-based tooling with hands-on examples.

#### Course Overview

1. **Module 1: Cloud Native Geospatial Fundamentals**
    - XArray Basics
    - STAC and Dask Basics
    - DuckDB Basics
    - Creating a Median Composite
2. **Module 2: Remote Sensing Fundamentals**
    - Calculating Spectral Indices
    - Masking Clouds
    - Extracting and Processing Time-Series
3. **Module 3: Computation and Data Processing**
    - Calculating Area
    - Computing Zonal Statistics
    - Interoperability with Google Earth Engine
4. **Module 4: Machine Learning and AI**
    - Supervised Classification with Embeddings
    - Unsupervised Clustering
5. **Module 5: Computation Environments**
    - Running Computation on Your Hardware
    - Using Google Cloud Runtime
    - Scaling Analysis in Cloud with Coiled
